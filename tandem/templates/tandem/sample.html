Sample Data

The TANDEM team used a test corpus to develop the application's functionality. The corpus, The Real Mother Goose by Blanch Fisher Wright (c1916), is a children's book in the public domain. As a result, we are to provide the content here as a learning tool.

To explore the functionality of TANDEM either download the full corpus (JPEGs) to run your own analysis in the app, or download our pre-processed CSV data output.

  


Using the Data

Creating a visualization can often help to contextualize meaning. Data visualizations can be created with a range of tools and in varying complexities. For instance, you can make a visualization that is as simple as a pie chart or a timeline with Microsoft Excel, or you can use a programming language to create something interactive.

Below we have provided examples of how to use the TANDEM data output for various visualization programs.

Excel

[click to enlarge]

Microsoft Excel is a proprietary platform that allows for quick and simple data manipulation. Google Sheets is a free alternative to Excel, but many people have a general familiarity with this tool. You can begin by using the built in charting functionality to select and visualize the tandemmain.csv. We have found that TANDEM data works well as bar charts, line charts, and circle graphs, but exploring other visualization styles in Excel will yield new and interesting perspectives. Your charts can be as simple or complex as you wish depending on the variables you’re interested. This image is a simple example of what can be done with Excel.

Data columns used for this visualization: Mean R, Mean G, Mean B, Total Word Count

Gephi

Gephi is a visualization environment that allows you to create complex network diagrams. The diagrams are both interactive and dynamic, reacting to user input or manipulation. Gephi makes it possible for you to find associations between objets as well as an understanding of the density or clustering of similar values. Use gephi to find connections in your data.

Wordle

[click to enlarge]

Wordle is a free online platform that allows you to create visualizations based word frequency. Using either the TandemAllText_allwords.csv or one of the page specific all_words/unique_words.csv’s, copy and paste all rows of data into Wordle. The platform will generate a word cloud. The larger the word, the more frequent it has occurred in the text. This is a common practice for text analysis and has become more mainstream thanks to platforms like this. *Java is required in order to run Wordle, but there are many word cloud alternatives emerging that achieve a similar result.

Data columns used for this visualization: TandemAllText_allwords.csv

R

R is a programming language used for statistical analysis and visualization. R has a diverse array of libraries built for graphics and data manipulation. This is the a highly versatile but alsos complex environments for those looking to explore the data created by our tool.

Data columns used for this visualization: Mean R, Word Frequency
