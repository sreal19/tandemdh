Best Practices

TANDEM is currently in its 0.5 phase, which means we're growing & testing. Below you will find some advice to get the best possible results from our application.

  • Files Types & Organization
  • Image Quality
  • Text Quality
  • Complying With Copyright

File Types & Organization

  • TANDEM can process TIFF, JPEG, and PDF file types.
    • Because PDF files require that the file name contain no spaces (use a dash instead).
  • We recommend you keep all related corpus files in one folder within your projects folder.
  • If you're processing more than one, save each page or image individually and in their original order.
    • Example: "MotherGoose1.jpg" and "MotherGoose2.jpg" will ensure that each file is processed sequentially.

Image Quality

  • If you are processing an image that includes color, make sure your images are in color (versus black and white scans).
  • Sometimes content retrieved from the open Web includes a watermark (such as from HathiTrust or Google Books). TANDEM's NLTK processor will pick up those watermarks. Consider cropping them out, or create your own scans instead.

Text Quality

  • TANDEM a has been quality tested for English language texts, and provides an output in ascii. If you analyze a corpus in another language, please double check the text output for accuracy.
  • Similarly, if you're corpus contains illuminated lettering (swirly, hand-drawn, or as part of an embedded image) double check the output to ensure NLTK picked up the text.

Complying With Copyright

Copyright can be a tricky subject, especially when materials are being used for scholarship or education. The TANDEM team would like to suggest erring on the side of caution, when possible.

  • We like you, and are glad you're using our application! But TANDEM is NOT responsible for your source material.
  • You are responsible for determining the copyright status of your source material. This means if you have processed scans from a book, you need to check the copyright before publishing or advertising the source files.
    • For instance, the files used for the TANDEM test corpus are available in the public domain, meaning that no single person holds the copyright for the material, which is how we are able to provide the Test Corpus and Sample Data.
