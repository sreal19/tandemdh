Sample Data

The TANDEM team used a test corpus to develop the application's functionality. The corpus, The Real Mother Goose by Blanch Fisher Wright (c1916), is a children's book in the public domain. As a result, we are to provide the content here as a learning tool.

To explore the functionality of TANDEM either download the full corpus (JPEGs) to run your own analysis in the app, or download our pre-processed CSV data output.

  


Using the Data

Creating a visualization can often help to contextualize meaning. Data visualizations can be created with a range of tools and in varying complexities. For instance, you can make a visualization that is as simple as a pie chart or a timeline with Microsoft Excel, or you can use a programming language to create something interactive.

Below we have provided examples of how to use the TANDEM data output for various visualization programs.
